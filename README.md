# Ignite Design System

Design System em React com tokens, componentes e documentação interativa via Storybook. Publicado no npm como `@igortullio-ui/react`.

## Demo

[Storybook](https://igortullio.github.io/ignite-05-challenge/?path=/story/home--page)

## Pacotes

| Pacote | Descrição |
|--------|-----------|
| `@igortullio-ui/tokens` | Design tokens (cores, fontes, espaçamentos, etc.) |
| `@igortullio-ui/react` | Componentes React (Avatar, Box, Button, Checkbox, Heading, MultiStep, Text, TextArea, TextInput) |
| `@igortullio-ui/eslint-config` | Configuração ESLint compartilhada |
| `@igortullio-ui/ts-config` | Configuração TypeScript compartilhada |
| `docs` | Documentação Storybook |

## Tecnologias

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Stitches](https://stitches.dev/)
- [Radix UI](https://www.radix-ui.com/)
- [Storybook](https://storybook.js.org/)
- [Changesets](https://github.com/changesets/changesets) para versionamento
- [tsup](https://tsup.egoist.dev/) para build

## Como executar

```bash
git clone https://github.com/igortullio/ignite-05-challenge.git
cd ignite-05-challenge
npm install
npm run dev
```
