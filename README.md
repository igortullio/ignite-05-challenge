# Ignite Design System

A React Design System with tokens, components, and interactive Storybook documentation. Published on npm as `@igortullio-ui/react`.

## Demo

[Storybook](https://igortullio.github.io/ignite-05-challenge/?path=/story/home--page)

## Packages

| Package | Description |
|---------|-------------|
| `@igortullio-ui/tokens` | Design tokens (colors, fonts, spacing, etc.) |
| `@igortullio-ui/react` | React components (Avatar, Box, Button, Checkbox, Heading, MultiStep, Text, TextArea, TextInput) |
| `@igortullio-ui/eslint-config` | Shared ESLint configuration |
| `@igortullio-ui/ts-config` | Shared TypeScript configuration |
| `docs` | Storybook documentation |

## Tech Stack

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Stitches](https://stitches.dev/)
- [Radix UI](https://www.radix-ui.com/)
- [Storybook](https://storybook.js.org/)
- [Changesets](https://github.com/changesets/changesets) for versioning
- [tsup](https://tsup.egoist.dev/) for bundling

## Getting Started

```bash
git clone https://github.com/igortullio/ignite-05-challenge.git
cd ignite-05-challenge
npm install
npm run dev
```
